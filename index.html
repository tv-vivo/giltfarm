Pasos para instalar la aplicación

  1. Abre la carpeta de descargas en tu dispositivo.
  2. Busca el archivo GiltFarm.apk.
  3. Haz clic en el archivo para comenzar la instalación.
  4. Si es necesario, activa la opción de "Orígenes desconocidos" en la configuración.
  5. Sigue las instrucciones en pantalla para completar la instalación.
